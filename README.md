# Demonstração do Algoritmo L*

## Arquivos Criados

1. **`requirements.txt`** - Arquivo texto com as bibliotecas e pré-requisitos listados.
2. **`demo_lstar_completo.py`** - Script Python completo para execução da demonstração

## Como Usar

### Executar Demonstração Completa
```
python demo_lstar_completo.py
```

## Pré-requisitos

Certifique-se de que as dependências estão instaladas:
```
pip install -r requirements.txt
```
